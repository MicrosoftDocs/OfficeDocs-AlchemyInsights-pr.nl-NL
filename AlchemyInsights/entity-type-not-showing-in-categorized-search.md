# <a name="entity-type-not-showing-in-search-results"></a>Entiteitstype niet wordt weergegeven in de zoekresultaten

Voor een entiteittype weer te geven in gecategoriseerde zoeken of tekst zoeken in de Dynamics 365 Unified Interface moet dat worden toegevoegd aan de module app.

Opmerking: Bij het zoeken naar bijlagen in de zoekopdracht relevant, de entiteit bijlage moet worden toegevoegd aan de module app zelfs als de bovenliggende record wat wordt weergegeven in de resultaten.
