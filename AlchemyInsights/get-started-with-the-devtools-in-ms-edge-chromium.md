# <a name="get-started-with-the-developer-tools-in-microsoft-edge-chromium"></a>Aan de slag met de hulpmiddelen voor ontwikkelaars in Microsoft Edge (chroom)

Microsoft Edge heeft het open-bronproject van chroom gemaakt en is bedoeld om het web minder gefragmenteerd en webplatformen beter compatibel te maken. Met deze wijziging kunt u Microsoft Edge gebruiken voor het maken en testen van uw websites die werken aan andere op chroom gebaseerde browsers.

Microsoft Edge (chroom) biedt ook de [beste hulpprogramma's voor ontwikkelaars](https://go.microsoft.com/fwlink/?linkid=2134941) die u bekend en compatibel met de hulpmiddelen die u al gebruikt.

Voor meer informatie raadpleegt u [nieuwe functies in de Microsoft Edge (chroom) devtools](https://go.microsoft.com/fwlink/?linkid=2135020).
