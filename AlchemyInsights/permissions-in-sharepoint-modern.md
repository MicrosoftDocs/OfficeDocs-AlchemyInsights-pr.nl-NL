# <a name="permissions-in-the-sharepoint-modern-experience"></a>Machtigingen in de moderne SharePoint-ervaring

Normaal zijn SharePoint-machtigingen beheerd via een set machtigingen groepen binnen een site (eigenaren, leden, bezoekers, enzovoort). In SharePoint in Microsoft 365 blijft dit waar voor sommige typen sites, maar er zijn meer opties beschikbaar.  

Ga voor meer informatie naar, [delen en machtigingen in de moderne SharePoint-ervaring](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions).
