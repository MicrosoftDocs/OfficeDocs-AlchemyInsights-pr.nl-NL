# <a name="edit-a-read-only-file"></a>Een alleen-lezen bestand bewerken

Wanneer u in documentbibliotheken op de naam van een document klikt, wordt het Office-programma mogelijk onverwacht geopend als een alleen-lezen bestand. Als u het wilt openen om te bewerken, klikt u op **openen** (de drie puntjes) en vervolgens op **bewerken.**
