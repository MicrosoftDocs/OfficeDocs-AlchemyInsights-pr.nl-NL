# <a name="edit-a-read-only-file"></a>Een alleen-lezen bestand bewerken

In documentbibliotheken, wanneer u op de naam van een document, kan het Office-programma onverwacht openen als alleen-lezen bestand. Klik op **openen** (de drie puntjes) om deze te openen voor bewerken, en klik op **bewerken**
