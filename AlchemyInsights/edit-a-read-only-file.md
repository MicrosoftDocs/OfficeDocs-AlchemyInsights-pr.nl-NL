# <a name="edit-a-read-only-file"></a>Een alleen-lezen bestand bewerken

In documentbibliotheken kan het Office-programma onverwacht openen als alleen-lezen bestand wanneer u op de naam van een document klikt. Als u het wilt openen voor bewerking, klikt u op **Openen** (de drie puntjes) en klikt u vervolgens op **Bewerken.**
