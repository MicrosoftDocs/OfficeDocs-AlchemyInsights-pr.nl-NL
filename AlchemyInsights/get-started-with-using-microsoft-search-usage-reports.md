# <a name="get-started-with-using-microsoft-search-usage-reports"></a>Aan de slag met het gebruiksrapporten van Microsoft Search

Gebruiksrapporten van Microsoft Search bieden inzicht in de werking van zoeken in uw organisatie. Met deze inzichten kunt u inhoud vinden en zoeken een handigere en aangenamere ervaring maken voor gebruikers in uw organisatie.

Zie Microsoft Search Usage Reports voor [meer informatie.](https://go.microsoft.com/fwlink/?linkid=2152048)
