# <a name="changing-public-folder-permissions"></a>Het wijzigen van machtigingen voor openbare mappen

Machtigingen voor openbare mappen kunnen worden gewijzigd door gebruikers en beheerders in Outlook. Beheerders kunnen ook machtigingen van de Exchange Admin Center (SBV) als volgt controleren:
  
1. IIn het Microsoft 365 admin center, gaat u naar de **Admin centers** \> **Exchange**.

2. Selecteer de **openbare mappen**.

3. Van daaruit kunt u machtigingen voor openbare mappen afzonderlijke door beveiligingsgroepen met machtigingen toe te wijzen. Voor een gebruiker machtigingen voor openbare mappen wijzigen, moet de gebruiker de eigendomsrechten hebben op de map.
