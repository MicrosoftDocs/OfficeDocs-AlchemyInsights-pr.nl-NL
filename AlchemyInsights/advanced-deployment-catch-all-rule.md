# <a name="deploy-office-apps"></a>Office apps implementeren

Voor het implementeren van Office apps aan meerdere gebruikers via de Office Deployment Tool, System Center Configuration Manager, als onderdeel van een installatiekopie van het besturingssysteem of in een gedeelde computer met behulp van Windows Server met Terminal of een extern bureaublad-Services (RDS) Services, kunt u overwegen de handleiding in de **Office 365 ProPlus implementatie advisor**.
  
Zorg ervoor dat u bent aangemeld als een huurder admin of globale admin en klik dan hier om te beginnen: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
