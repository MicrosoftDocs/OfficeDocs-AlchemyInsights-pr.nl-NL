# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>Meer informatie over het toewijzen van machtigingen tussen Google Chrome-beleidsregels en Microsoft Edge-beleid

Google Chrome-beleidsregels worden toegewezen aan de relevante Microsoft Edge-beleidsregels die worden ondersteund in versie 80 van de browser. Voor meer informatie raadpleegt [u Google Chrome to Microsoft Edge Policy mapping](https://go.microsoft.com/fwlink/?linkid=2141933).
