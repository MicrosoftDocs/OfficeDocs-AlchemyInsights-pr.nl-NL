# <a name="run-microsoft-edge-in-kiosk-mode"></a>Microsoft Edge uitvoeren in kioskmodus

U kunt kioskmodus configureren in Microsoft Edge-versies 77 en hoger, evenals in de oudere versie van Microsoft Edge 45 en eerder.

U kunt apparaten die worden uitgevoerd in Windows 10 configureren voor gebruik in de kioskmodus met één app of voor [meerdere apps.](https://go.microsoft.com/fwlink/?linkid=2133659)

Zie voor meer informatie over het instellen van kioskmodus in de nieuwste versie van het stabiele Microsoft Edge-kanaal en in de oudere versie van Microsoft Edge:

- [Microsoft Edge met toegewezen toegang](https://docs.microsoft.com/deployedge/microsoft-edge-configure-kiosk-mode#microsoft-edge-with-assigned-access)
- [Opdrachtregelparameter Microsoft Edge -kiosk](https://answers.microsoft.com/microsoftedge/forum/msedge_open-msedge_win10/access-microsoft-edge-using-command-line/03a4add6-9ca4-4fbb-a183-aaa763a0ab76)
- [Ondersteuning voor de verouderde kioskmodus van Microsoft Edge](https://blogs.windows.com/msedgedev/2021/02/05/what-you-need-to-know-about-kiosk-mode-when-support-for-microsoft-edge-legacy-ends/)
