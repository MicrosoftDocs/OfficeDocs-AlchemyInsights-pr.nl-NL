# <a name="live-events-in-microsoft-stream"></a>Livegebeurtenissen in Microsoft Stream

U kunt livegebeurtenissen maken met Microsoft Stream in de hele organisatie. U kunt livegebeurtenissen plannen, produceren en leveren voor diverse scenario's, zoals bedrijfsevenementen, leiderschapsupdates en meer.

- [Een livegebeurtenis maken in Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Aan de slag met livegebeurtenissen van Microsoft Teams](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Coderingen configureren voor livestreaming in Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
