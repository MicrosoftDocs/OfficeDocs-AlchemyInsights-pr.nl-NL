# <a name="live-events-in-microsoft-stream"></a>Live gebeurtenissen in Microsoft stream

U kunt live gebeurtenissen maken met Microsoft stream binnen de organisatie. U kunt live gebeurtenissen plannen, produceren en geven voor tal van scenario's, zoals gebeurtenissen in het hele bedrijf, leiderschaps updates en meer.

- [Een live gebeurtenis maken in Microsoft stream](https://docs.microsoft.com/stream/live-create-event)
- [Aan de slag met Microsoft teams Live gebeurtenissen](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Coderingsprogramma's voor live streaming configureren in Microsoft stream](https://docs.microsoft.com/stream/live-encoder-setup)
