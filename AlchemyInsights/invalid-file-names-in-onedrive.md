# <a name="invalid-file-and-folder-names-in-onedrive-and-sharepoint"></a>Ongeldige namen voor bestanden en mappen in OneDrive en SharePoint

De volgende tekens zijn niet toegestaan in namen voor bestanden en mappen \* : \< \> ? / \ | 
  
Sommige organisaties hebben nog geen ondersteuning voor # en% ingeschakeld. Zie [ondersteuning voor # en%](https://go.microsoft.com/fwlink/?linkid=862611)in-en uitschakelen voor meer informatie over het toestaan van deze tekens in uw organisatie.
