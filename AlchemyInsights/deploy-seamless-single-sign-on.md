# <a name="enable-seamless-sso"></a>Naadloze SSO inschakelen

Azure Active Directory (Azure AD) Seamless Single Sign-On (Seamless SSO) meldt zich automatisch aan bij gebruikers wanneer ze zich op hun bedrijfsdesktops bevinden die zijn verbonden met uw bedrijfsnetwerk.
  
[Volg de stappen in dit artikel om Seamless SSO in te schakelen](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
