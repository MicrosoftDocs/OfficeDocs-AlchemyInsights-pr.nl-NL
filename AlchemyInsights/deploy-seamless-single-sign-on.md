# <a name="enable-seamless-sso"></a>Naadloze eenmalige aanmelding inschakelen

Azure Active Directory (Azure AD) naadloze eenmalige aanmelding (naadloze SSO) ondertekent automatisch in gebruikers wanneer ze zich op hun zakelijke bureaubladen die zijn verbonden met uw bedrijfsnetwerk.
  
[Volg de stappen in dit artikel om in te schakelen naadloze eenmalige aanmelding](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
