# <a name="change-the-font-size-in-a-message"></a>De tekengrootte in een bericht wijzigen

1. Als u het bericht al hebt getypt, selecteert u de tekst.
2. Selecteer het  **opmaak** pictogram onder het bericht (dit is het eerste pictogram).
3. Selecteer het pictogram voor de  **tekengrootte**  boven het bericht (dit is het zevende pictogram).
4. Selecteer een formaat.
