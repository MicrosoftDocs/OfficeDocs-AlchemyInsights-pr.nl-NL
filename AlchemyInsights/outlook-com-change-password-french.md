# <a name="changer-votre-mot-de-passe-dans-outlookcom"></a>Wisselaar votre mot de Passe dans Outlook.com

Votre mot de Passe Outlook.com est le même que votre mot de Passe Compte Microsoft.

1. Accédez à [https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409) et Sélectionnez **Sécurité du mot de Passe**.
2. Par mesure de Sécurité, vous pouvez être invité à confirmr votre identité à l'aide d'un code de Sécurité.
3. Entrez votre mot de Passe Actuel, Votre nouveau mot de passe, puis Sélectionnez **Enregistrer**.

Si vous avez oublié votre mot de passe, accédez à la [Page The réinitialisation de mot de Passe de Compte Microsoft](https://go.microsoft.com/fwlink/p/?linkid=841909), puis Suivez van de uitnodigingen.

Si vous Pensez que votre Outlook.com een été piraté, Voir [ma compte Outlook.com een été piraté](https://support.office.com/fr-fr/article/mon-compte-outlook-com-a-été-piraté-35993ac5-ac2f-494e-aacb-5232dda453d8?wt.mc_id=Office_Outlook_com_Alchemy).
