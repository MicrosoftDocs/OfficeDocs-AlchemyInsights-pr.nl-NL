# <a name="how-to-create-a-file-request"></a>Een bestandsaanvraag maken

Met de functie voor bestandsaanvragen in OneDrive kun je een map kiezen waarin anderen bestanden kunnen uploaden via een link die je ze stuurt. De gebruikers van wie u bestanden aanvraagt, kunnen niet zien wat er in de map zit--ze kunnen alleen bestanden uploaden.

[Een bestandsaanvraag maken](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af)
