# <a name="configure-privilege-for-user-and-system-chart"></a>Bevoegdheden configureren voor gebruikers- en systeemdiagram

**Symptoom:** Geen leesvoorrecht voor een grafiek.

**Oorzaak:** Niet voldoende bevoegdheden of bevat geen benodigde bevoegdhedenfouten.

**Oplossing:** Bevoegdheden configureren voor grafiek

**Stappen**:

1. Ga naar **Instellingen -> Aanpassen -> Het systeem aanpassen -> beveiligingsrollen**.

2. Dubbelklik op de rol.

    - Configuratie van gebruikersdiagramrechten staat onder het **tabblad Core Records.**
    - Configuratie van systeemdiagramrechten staat onder het **tabblad** Aanpassen.
