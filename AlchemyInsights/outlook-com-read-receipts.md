# <a name="read-receipts"></a>Leesbevestigingen

U kunt geen een leesbevestiging in Outlook.com, maar u kunt leesbevestigingen voor Outlook.com-berichten die worden verzonden met Outlook voor Windows aanvragen. Voor meer informatie over hoe, Zie [aanvraag ontvangsten en levering-berichten lezen en toevoegen](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141).
  
Om te kiezen hoe Outlook.com reageert op leesbevestigingen, Ga naar [Instellingen voor ontvangstbewijzen lezen](https://outlook.live.com/mail/options/mail/handling/readReceipts).
