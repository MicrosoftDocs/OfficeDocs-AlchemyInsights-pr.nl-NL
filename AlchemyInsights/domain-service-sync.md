# <a name="domain-service-synchronization"></a>Domein service-synchronisatie

Objecten en referenties in een Azure AD DS-domein (Active Directory Domain Services) van Azure Active Directory Domain Services (Azure AD DS) kunnen lokaal binnen het domein worden gemaakt of worden gesynchroniseerd vanuit een Azure Active Directory (Azure AD)-Tenant. Wanneer u Azure AD DS voor het eerst implementeert, wordt een automatische synchronisatie van eenrichtingssynchronisatie geconfigureerd en gestart met het repliceren van objecten vanuit Azure AD. Deze eenrichtingssynchronisatie wordt uitgevoerd op de achtergrond om het Azure AD DS-domein up-to-date te houden met wijzigingen in azure AD. Er vindt geen synchronisatie plaats van Azure AD DS terug in azure AD.

Zie [Domain Service Synchronization (domein service Synchronization](https://docs.microsoft.com/azure/active-directory-domain-services/synchronization)) voor meer informatie over het synchroniseren van Azure Active Directory Domain Services.
