# <a name="microsoft-defender-atp-missed-a-malicious-item"></a>Microsoft Defender ATP miste een kwaadaardig item

- Dien het schadelijke bestand bij Microsoft voor analyse op de [Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/filesubmission/). 
- Als u bent aangemeld, zoekt u de inzendingsgeschiedenis voor een bestand op de [Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/submissionhistory).
- Zie [Bestanden voor analyse verzenden voor](https://docs.microsoft.com/windows/security/threat-protection/intelligence/submission-guide)meer informatie.
