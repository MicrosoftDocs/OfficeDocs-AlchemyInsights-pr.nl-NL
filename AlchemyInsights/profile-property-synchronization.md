# <a name="profile-property-synchronization"></a>Synchronisatie van profieleigenschappen

U kunt geen extra eigenschappen toewijzen voor AD-import om te synchroniseren vanuit Azure Active Directory naar de Gebruikersprofieltoepassing. Het importeren van AD-gegevens is beperkt tot een vooraf geconfigureerde set eigenschappen, zodat de prestaties van de timeropdracht consistent zijn.
  
[Meer informatie over de profieleigenschappen die worden gesynchroniseerd](https://go.microsoft.com/fwlink/?linkid=875671)
