# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Openbare map verwijderd uit Outlook favorieten op het web (OWA)

Er is een bekend probleem met het verwijderen van openbare mappen uit favorieten in OWA bij elke aanmelding of schermvernieuwing. Het probleem wordt binnenkort opgelost. Tot die tijd voegt u de map opnieuw toe voor toegang vanuit OWA.
