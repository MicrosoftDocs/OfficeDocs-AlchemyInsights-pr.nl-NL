# <a name="how-to-create-a-rule-in-outlook-for-mac"></a>Een regel maken in Outlook voor Mac

1. Als u een map wilt maken voor de e-mailberichten die door de regel worden verzonden, selecteert u het hoofdpostvak, klikt u op het tabblad **indelen** op **nieuwe map**, typt u de naam van de map en drukt u op ENTER.

2. Klik in de berichtenlijst op een e-mailbericht waarvoor u een regel wilt maken.

3. Klik op het tabblad **Start** op **regels**en klik vervolgens op **regel maken**.

4. Klik onder **Wanneer een nieuw bericht binnenkomt** op aan uw wensen voldoet. 

Zie [een regel maken in Outlook voor Mac](https://aka.ms/AA1uy0v) voor gedetailleerde instructies voor het maken van aangepaste regels.
