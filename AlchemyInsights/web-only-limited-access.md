# <a name="web-only-limited-access"></a>Web-Only beperkte toegang

Web-Only beperkte toegang is een variant van het machtigingsniveau beperkte toegang, zodat gebruikers alleen toegang hebben tot het webobject.

Zie [standaardmachtigingsniveaus](https://docs.microsoft.com/sharepoint/understanding-permission-levels#default-permission-levels)voor meer informatie.
