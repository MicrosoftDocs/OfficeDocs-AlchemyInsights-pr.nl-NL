# <a name="generate-automatic-captions-and-a-transcript-in-microsoft-stream"></a>Automatische bijschriften en een transcriptie genereren in Microsoft Stream

Stream kan automatisch bijschriften genereren met behulp van automatische spraakherkenningstechnologie en u kunt een schuiftranscriptievenster weergeven om eenvoudig te navigeren en te volgen, samen met de gesproken inhoud van een video.

- [Automatisch ondertitels en een transcriptie genereren voor je Microsoft Stream-video's](https://docs.microsoft.com/stream/portal-autogenerate-captions)

- [Een schuiftranscriptievenster voor een video in Microsoft Stream laten zien](https://docs.microsoft.com/stream/portal-configure-transcript-mode)
