# <a name="solutions-for-unsupported-operating-system"></a>Oplossingen voor "Niet-ondersteund besturingssysteem"

Wanneer u Office 365 installeren probeert, verschijnt er een foutbericht **niet-ondersteund besturingssysteem** als uw apparaat niet voldoet aan de [systeemvereisten](https://products.office.com/office-system-requirements).
  
Office 365 is vereist:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 servicepack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, of Windows Server 2008 R2

    OF

- Mac OS X 10.10

Als dit bericht tijdens het installeren van Office op een mobiel apparaat of een Chromebook, Zie [het installeren en uitvoeren van Microsoft Office op een Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
