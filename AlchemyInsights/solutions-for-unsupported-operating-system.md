# <a name="solutions-for-unsupported-operating-system"></a>Oplossingen voor "Niet-ondersteund besturingssysteem"

Wanneer u Microsoft 365 probeert te installeren, ziet u mogelijk een foutmelding voor het **besturingssysteem** zonder ondersteuning als uw apparaat niet aan de [systeemvereisten](https://products.office.com/office-system-requirements)voldoet.
  
Microsoft 365 vereist:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 of Windows Server 2008 R2

    OF

- Mac OS X 10.10

Als dit bericht is ontvangen terwijl u Office probeert te installeren op een mobiel apparaat of een Chromebook, raadpleegt u [Microsoft Office installeren en uitvoeren op een Chromebook.](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA)
