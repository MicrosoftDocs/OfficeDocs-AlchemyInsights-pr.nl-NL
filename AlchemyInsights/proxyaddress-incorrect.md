# <a name="proxyaddress-incorrect"></a>ProxyAddress onjuist

Wanneer een object wordt gesynchroniseerd met Azure AD, worden de waarden die zijn opgegeven in het kenmerk proxyAdressen in Active Directory vergeleken met Azure AD-regels en wordt het kenmerk proxyAdressen ingevuld in Azure AD. Daarom zijn de waarden van het kenmerk proxyAdressen voor het object in Active Directory mogelijk niet dezelfde als de waarden van het kenmerk proxyAdressen in Azure AD.
  
Zie [Hoe het kenmerk proxyAddress wordt ingevuld in Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad)voor meer informatie over hoe het proxyadres wordt ingevuld.
