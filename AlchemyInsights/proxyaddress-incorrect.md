# <a name="proxyaddress-incorrect"></a>Onjuiste ProxyAddress

Wanneer een object wordt gesynchroniseerd naar Azure AD, de waarden die zijn opgegeven in het kenmerk proxyAddresses in Active Directory worden vergeleken met Azure AD-regels, en vervolgens het kenmerk proxyAddresses in Azure AD is gevuld. Dus de waarden van het kenmerk proxyAddresses voor het Active Directory-object mogelijk niet hetzelfde als de waarden van het kenmerk proxyAddresses in Azure AD.
  
Meer informatie over hoe de proxyaddress is gevuld, Zie [hoe het kenmerk proxyAddress in Azure AD wordt gevuld](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
