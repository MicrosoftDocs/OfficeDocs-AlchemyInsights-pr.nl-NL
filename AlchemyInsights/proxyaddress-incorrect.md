# <a name="proxyaddress-incorrect"></a>Onjuiste ProxyAddress

Wanneer een object wordt gesynchroniseerd met Azure AD, worden de waarden die zijn opgegeven in het kenmerk proxyAddresses in Active Directory vergeleken met de Azure AD-regels en wordt het proxyAddresses-kenmerk ingevuld in azure AD. De waarden van het kenmerk proxyAddresses voor het object in Active Directory kunnen daarom niet gelijk zijn aan de waarden van het proxyAddresses-kenmerk in azure AD.
  
Als u meer wilt weten over de manier waarop de proxyAddress is ingevuld, raadpleegt u [hoe het proxyAddress-kenmerk in azure AD is ingevuld](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
