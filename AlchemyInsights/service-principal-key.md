# <a name="service-principal-key"></a>Service-Principal-sleutel

Zie de volgende artikelen voor meer informatie over het beheren van belangrijkste referenties voor een Service-Principal:

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) -voor het aanvragen van sleutel referenties voor een service-principal met behulp van PowerShell.

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) : voor het maken van een nieuwe sleutel referentie voor een service-principal met behulp van PowerShell.

- [Remove-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) -om een sleutel referentie te verwijderen uit een service-principal met behulp van PowerShell.
