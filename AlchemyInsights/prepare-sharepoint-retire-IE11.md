# <a name="prepare-your-sharepoint-environment-for-the-retirement-of-internet-explorer-11"></a>Uw omgeving SharePoint voorbereiden op de uittreding van Internet Explorer 11

Hoewel de ondersteuning voor Internet Explorer 11 blijft bestaan, hebben we de ontwikkelingsbronnen voor webbrowsers verplaatst naar [de Microsoft Edge browser.](https://www.microsoft.com/edge/business) 

Zie Uw omgeving voorbereiden SharePoint voor de uittreding van [Internet Explorer 11](/sharepoint/prepare-ie11)voor Microsoft 365 apps en services voor meer informatie.
