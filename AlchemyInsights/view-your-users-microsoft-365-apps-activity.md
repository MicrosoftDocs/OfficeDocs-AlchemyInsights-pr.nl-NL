# <a name="view-your-users-microsoft-365-apps-activity"></a>De activiteit van uw gebruikers in Microsoft 365-apps weergeven

Het [rapport Office-activering](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide) geeft u een weergave van welke gebruikers hun Microsoft 365-apps op ten minste één apparaat hebben geïnstalleerd en geactiveerd.

Bovendien u een overzicht krijgen van de Microsoft 365 Apps-activiteit van uw gebruikers met behulp van het [gebruiksrapport Microsoft 365 Apps.](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide)
