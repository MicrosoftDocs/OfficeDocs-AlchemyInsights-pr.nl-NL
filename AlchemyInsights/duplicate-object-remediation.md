# <a name="duplicate-object-synchronization-errors"></a>Fouten met synchronisatie van objecten dupliceren

Als u een van de volgende fouten tijdens een synchronisatie ontvangt:

- De Microsoft Online Services-map bestaat al met een gesynchroniseerd object dat hetzelfde proxyadres bevat.

- Een vergelijkbaar bericht voor UserPrincipalName.

Gebruik de nieuwe AADConnect status diagnostisch om dubbele object fouten op te lossen. Zie voor meer informatie [problemen met gedupliceerde kenmerk synchronisatie oplossen en herstellen](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
