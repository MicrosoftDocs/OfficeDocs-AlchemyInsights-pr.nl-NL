# <a name="microsoft-365-tax-information"></a>Belastinginformatie Microsoft 365

De belastingen voor aankopen in het Microsoft 365-beheercentrum die u maakt via Microsoft, worden bepaald door uw bedrijfsadres of, indien dat verschilt, door uw verzendadres. Zie [Belastinginformatie](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/tax-information?view=o365-worldwide)voor een volledig overzicht.

- [Welke belasting moet u betalen?](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/tax-information?view=o365-worldwide#what-tax-will-i-be-charged) 

- [Hoe worden de belastingen berekend?](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/tax-information?view=o365-worldwide#how-taxes-are-calculated)

- [Belastingvrijstelling aanvragen](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/tax-information?view=o365-worldwide#apply-for-tax-exempt-status).
