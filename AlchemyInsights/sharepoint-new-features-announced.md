# <a name="sharepoint-new-features-announced"></a>SharePoint nieuwe functies aangekondigd

Op de Ignite 2018-conferentie kondigde Microsoft nieuwe innovaties voor SharePoint aan, waaronder nieuwe functies voor moderne portals, nieuwe webonderdelen, tal van verbeteringen aan hubsites, nieuwskoppeling, doelgroeptargeting, beschikbaarheid van nieuwe mobiele apps, verbinding met Teams en meer...
  
Zie de blogpost Nieuwe innovaties [in SharePoint &amp; Microsoft 365 bieden kracht en eenvoud voor samenwerking met inhoud voor](https://go.microsoft.com/fwlink/?linkid=2026502) meer informatie over deze functies.
  
Uw organisatie kan deelnemen aan het Programma Gerichte release om vroegtijdig toegang te krijgen tot nieuwe functies. Zie [De opties voor standaard of gerichte release instellen in Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/manage/release-options-in-office-365).
