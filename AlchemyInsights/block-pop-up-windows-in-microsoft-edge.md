# <a name="block-pop-up-windows-in-microsoft-edge"></a>Pop-upvensters blokkeren in Microsoft Edge

1. Gaat u naar **... (Instellingen en meer)**  >  **Instellingen**  >  voor **Site machtigingen**.
2. Selecteer **pop-ups en omleidingen**.
3. Zet **blok**.
