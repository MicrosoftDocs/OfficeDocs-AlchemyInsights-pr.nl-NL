# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Een gedeeld postvak gebruiken op een mobiel apparaat

Als u een gedeelde postbus op uw mobiele apparaat wilt gebruiken, moet u [Outlook Mobile downloaden](https://products.office.com/outlook-mobile-for-android-and-ios) en vervolgens [een gedeeld postvak toevoegen aan Outlook Mobile](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f).
  
Hulp krijgen: [in-app-Help opvragen voor Outlook voor IOS en Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
