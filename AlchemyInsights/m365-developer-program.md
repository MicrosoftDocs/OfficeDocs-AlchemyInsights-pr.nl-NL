# <a name="microsoft-365-developer-program"></a>Microsoft 365 Ontwikkelaarsprogramma

Het Microsoft 365 Developer Program bevat een Microsoft 365 E5 ontwikkelaarsabonnement waarmee u uw eigen sandbox kunt maken en oplossingen kunt ontwikkelen, onafhankelijk van uw productieomgeving. Zie voor meer informatie en veelgestelde vragen over het Microsoft 365 Developer Program en Visual Studio-abonnementen [Microsoft 365 Developer Program.](/office/developer-program/microsoft-365-developer-program)

Voor hulp bij het Microsoft 365 Developer Program of Visual Studio abonnementen, dient u een ondersteuningsaanvraag in [Visual Studio Abonnementsondersteuning](https://visualstudio.microsoft.com/subscriptions/support/).
