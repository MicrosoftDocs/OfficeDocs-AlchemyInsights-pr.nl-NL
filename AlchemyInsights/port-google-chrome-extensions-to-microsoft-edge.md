# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>Google Chrome-uitbreidingen voor poorten van Microsoft Edge (chroom)

[Google Chrome-extensies kunt u eenvoudig overbrengen naar Microsoft Edge (chroom)](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension). In de meeste gevallen zijn er alleen minimale wijzigingen nodig om deze extensies op Microsoft Edge uit te voeren.

De Api's met extensies en de manifest Keys die worden ondersteund door Google Chrome zijn code compatibel met Microsoft Edge. Microsoft Edge biedt echter geen ondersteuning voor de extensie Api's Chrome. GCM, Chrome. Identity. getAccounts, Chrome. Identity. getAuthToken en Chrome. instanceID.
