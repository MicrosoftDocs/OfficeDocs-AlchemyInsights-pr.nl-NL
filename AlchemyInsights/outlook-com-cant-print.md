# <a name="unable-to-print"></a>Kan niet afdrukken

We zijn op de hoogte van een probleem waarbij sommige gebruikers niet kunnen afdrukken wanneer ze gebruikmaken van een Safari-of Edge-browser. Gebruik een andere browser om dit probleem te verhelpen. Onze excuses voor het ongemak.
