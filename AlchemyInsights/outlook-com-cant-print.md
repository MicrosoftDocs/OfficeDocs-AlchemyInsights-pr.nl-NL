# <a name="unable-to-print"></a>Kan niet afdrukken

We zijn op de hoogte van een probleem waarbij sommige gebruikers niet kunnen afdrukken wanneer ze Safari of Edge browsers gebruiken. Terwijl we werken aan een oplossing hiervoor, gebruik dan een andere browser. Wij verontschuldigen ons voor het ongemak.
