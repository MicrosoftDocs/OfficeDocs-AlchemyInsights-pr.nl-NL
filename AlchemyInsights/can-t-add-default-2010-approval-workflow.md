# <a name="cant-add-default-2010-approval-workflow"></a>Kan standaard 2010 goedkeuringsworkflow niet toevoegen

In een Microsoft SharePoint-siteverzameling u een globaal herbruikbare werkstroom (zoals ' goedkeuring-SharePoint 2010 ') niet toevoegen aan een lijst of bibliotheek. De goedkeuringsworkflow ontbreekt mogelijk. Raadpleeg [een globaal herbruikbare werkstroom toevoegen aan een SharePoint-lijst of-bibliotheek](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library)om dit probleem op te lossen.

Zie [Alles over goedkeuringsworkflows](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742)voor meer informatie over Workflowgoedkeuringen. 
 
Meer informatie over [moderne goedkeuringen in flow](https://flow.microsoft.com/blog/introducing-modern-approvals).
