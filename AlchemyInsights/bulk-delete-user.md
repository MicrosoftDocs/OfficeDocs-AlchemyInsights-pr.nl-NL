# <a name="bulk-delete-user"></a>Bulk verwijderen gebruiker

U een groot aantal gebruikers uit de groep verwijderen door een CSV-bestand (Comma separated Values) te gebruiken om gebruikers in bulk te verwijderen. Zie Gebruikers [voor bulkverding verwijderen in Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete)als u bulkgebruikers wilt verwijderen met Azure AD-portal.
