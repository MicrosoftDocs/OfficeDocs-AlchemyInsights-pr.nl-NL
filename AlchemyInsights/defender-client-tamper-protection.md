# <a name="defender-tamper-protection"></a>Defender sabotagebeveiliging 

Met sabotagebeveiliging, kwaadaardige apps worden verhinderd om acties zoals deze te nemen:

- Virus-en bedreigingsbeveiliging uitschakelen
- Real-time beveiliging uitschakelen
- Gedragsbewaking uitschakelen
- Antivirus uitschakelen (zoals Iofficeanti virus (IOAV))
- Door de Cloud geleverde beveiliging uitschakelen
- Updates van beveiligingsinformatie verwijderen

Zie [Beveiligingsinstellingen beveiligen met sabotagebeveiliging](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/prevent-changes-to-security-settings-with-tamper-protection)voor meer informatie.
