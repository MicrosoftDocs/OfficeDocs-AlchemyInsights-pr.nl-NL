# <a name="learn-about-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Meer informatie over het toewijzen van oudere beleidsregels en Microsoft Edge-beleidsregels voor Microsoft Edge

Groepsbeleidsregels voor Microsoft Edge Legacy (versies 45 en eerder) en de MDM-instellingen (Mobile Device Management) zijn toegewezen aan de beleidsregels van Microsoft Edge versie 80. Zie [Microsoft Edge verouderd aan Microsoft Edge-beleidstoewijzingen](https://go.microsoft.com/fwlink/?linkid=2141665)voor meer informatie.
