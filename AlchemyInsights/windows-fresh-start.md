# <a name="windows-fresh-start"></a>Windows Fresh Start

Windows Fresh Start-apparaatactie biedt een alternatief voor de acties Device Wipe en Device Retire, samen met de optie om de apparaatinschrijvingsstatus te behouden, samen met de apps en instellingen voor de thuismap van de gebruiker.

Zie [Nieuwe start gebruiken om Windows 10-apparaten opnieuw in te stellen met Intune](https://docs.microsoft.com/intune/device-fresh-start).
