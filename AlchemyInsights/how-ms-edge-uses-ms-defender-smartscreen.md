# <a name="how-microsoft-edge-uses-microsoft-defender-smartscreen"></a>Hoe Microsoft Edge gebruikmaakt van Microsoft Defender SmartScreen

Om u veilig te houden, wordt Microsoft Edge Microsoft voor perpetrate gebruikt als een vroeg waarschuwingssysteem tegen websites die phishing-aanvallen beschermen en malware distribueren via gerichte aanvallen.

Beheerders kunnen Microsoft Defender SmartScreen configureren met behulp van Groepsbeleid, Microsoft intune of MDM-instellingen (Mobile Device Management). Op basis van het instellen van Microsoft Defender SmartScreen kunt u gebruikers een waarschuwingspagina weergeven en ze de site laten doorlopen of de site helemaal blokkeren.

Voor meer informatie raadpleegt u [Microsoft edge support voor Microsoft Defender SmartScreen](https://go.microsoft.com/fwlink/?linkid=2133081) en [Microsoft Defender Application Guard](https://go.microsoft.com/fwlink/?linkid=2132839).
