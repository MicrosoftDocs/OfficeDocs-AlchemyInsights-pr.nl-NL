# <a name="microsoft-365-multi-geo"></a>Microsoft 365 Multi-Geo

Met Microsoft 365 Multi-Geo kan uw organisatie haar Microsoft 365-aanwezigheid uitbreiden naar meerdere geografische regio's en/of landen binnen uw bestaande tenant. Zie de onderstaande links voor meer informatie.

- [Microsoft 365 Multi-Geo](https://docs.microsoft.com/office365/enterprise/office-365-multi-geo)
- [Multigeo-mogelijkheden in OneDrive en SharePoint Online](https://docs.microsoft.com/office365/enterprise/multi-geo-capabilities-in-onedrive-and-sharepoint-online-in-office-365)
- [Multi-Geo-mogelijkheden in Exchange Online](https://docs.microsoft.com/office365/enterprise/multi-geo-capabilities-in-exchange-online)
- [Plan voor Microsoft 365 Multi-Geo](https://docs.microsoft.com/office365/enterprise/plan-for-multi-geo)
- [Multigeo configureren](https://docs.microsoft.com/office365/enterprise/multi-geo-tenant-configuration)
