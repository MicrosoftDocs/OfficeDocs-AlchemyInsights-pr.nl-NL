# <a name="set-passwords-to-never-expire"></a>Set wachtwoorden nooit verlopen.

Instellen dat wachtwoorden nooit verlopen van [Security &amp; Privacy](https://portal.office.com/adminportal/home#/settings/security). Klik op 'Bewerken' naast het wachtwoordbeleid en zet de schuifregelaar voor "uit".
  
Zie voor meer gedetailleerde informatie: [instellen van het wachtwoordbeleid voor uw organisatie](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
