# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Ondersteunde versie van het SharePoint-migratieprogramma



Het SharePoint-migratieprogramma ondersteunt momenteel alleen migraties van SharePoint 2013. Versies 2010 en 2016 worden niet ondersteund op dit moment.
  
Zie [introductie van het SharePoint-migratieprogramma](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409)voor meer informatie.
