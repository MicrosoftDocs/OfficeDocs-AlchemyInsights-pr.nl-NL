# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Ondersteunde versie van het hulpprogramma SharePoint Migration

Het hulpprogramma SharePoint Migration ondersteunt momenteel alleen migraties vanuit SharePoint 2013. 2010 versies en 2016 worden niet ondersteund op dit moment.
  
Zie voor meer informatie, de [Introductie van het hulpprogramma SharePoint Migration](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
