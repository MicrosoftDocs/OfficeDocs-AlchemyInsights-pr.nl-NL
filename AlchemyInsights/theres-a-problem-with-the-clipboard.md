# <a name="resolving-excel-clipboard-error"></a>Fout bij het oplossen van het probleem in Excel

Als een foutbericht wordt weergegeven met de tekst ' er is een probleem opgetreden met het Klembord, maar u kunt nog steeds de inhoud in deze werkmap plakken, ' Probeer de functie voor livevoorbeeld uit te schakelen door de volgende stappen uit te voeren:

1. Klik in Excel op **Bestands**  >  **Opties**.
3. Selecteer in het dialoogvenster **Opties voor Excel** de optie **Algemeen**.
4. Schakel onder Opties voor de **gebruikers interface**de optie **livevoorbeeld inschakelen**uit.
5. Klik op **OK**.
6. Start Excel opnieuw.
