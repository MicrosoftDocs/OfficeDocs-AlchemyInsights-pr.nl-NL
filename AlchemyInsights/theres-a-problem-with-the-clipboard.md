# <a name="disable-excel-live-preview"></a>Excel Live voorvertoning uitgeschakeld

Als er een in Excel dat zegt foutbericht: "Er is een probleem met het Klembord, maar u kunt nog steeds de inhoud van deze werkmap plakken" kunt u de functie Livevoorbeelden uitschakelen door de volgende stappen:

1. In Excel, klikt u op het **bestand** > **Opties**.
3. Selecteer **Algemeen**in het dialoogvenster **Opties voor Excel** .
4. Schakel onder **Opties voor gebruikersinterface** **Live voorvertoning inschakelen**.
5. Klik op **OK**.
6. Start Excel opnieuw.
