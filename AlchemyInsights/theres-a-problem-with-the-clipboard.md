# <a name="resolving-excel-clipboard-error"></a>Excel-klembordfout oplossen

Als u een foutmelding krijgt in Excel met de tekst 'Er is een probleem met het klembord, maar u kunt uw inhoud nog steeds in deze werkmap plakken', kunt u de functie Live Preview uitschakelen door de volgende stappen uit te voeren:

1. Klik in Excel op  >  **Bestandsopties.**
3. Selecteer algemeen in het dialoogvenster Opties **voor** **Excel.**
4. Schakel **onder Gebruikersinterfaceopties** de optie **Livevoorbeeld inschakelen uit.**
5. Klik op **OK**.
6. Start Excel opnieuw.
