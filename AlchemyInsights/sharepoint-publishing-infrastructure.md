# <a name="sharepoint-publishing-infrastructure"></a>SharePoint-publicatie infrastructuur


SharePoint bevat een klassieke reeks functies waarmee u uitgebreide webpagina's voor uw intranet maken en publiceren. Activeer de SharePoint-publicatie infrastructuur op het niveau van de siteverzameling om deze in te schakelen.

Voor meer informatie u terecht op de onderstaande links.

[Publicatiefuncties inschakelen](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Publicatiefuncties](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Notities:

- De klassieke publicatiesite sjablonen bevatten standaard de publicatiefuncties.

- Klassieke publicatiefuncties zijn niet beschikbaar op moderne sites. Moderne sites bieden standaard een nieuwer alternatief.
