# <a name="sharepoint-publishing-infrastructure"></a>Publicatie-infrastructuur van SharePoint


SharePoint bevat een klassieke set functies waarmee u webpagina's kunt maken en publiceren voor uw intranet. Activeer de SharePoint-publicatie-infrastructuur op het niveau van de siteverzameling om ze in te schakelen.

Voor meer informatie raadpleegt u de onderstaande koppelingen.

[Publicatiefuncties inschakelen](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Publicatiefuncties](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Opmerkingen:

- De klassieke publicatiesite sjablonen bevatten standaard de publicatiefuncties.

- Klassieke publicatiefuncties zijn niet beschikbaar voor moderne sites. Moderne sites geven standaard een nieuwer alternatief.
