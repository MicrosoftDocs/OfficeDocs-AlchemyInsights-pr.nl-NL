# <a name="delete-a-communication-site"></a>Een communicatiesite verwijderen

1. Ga naar de site.
    
2. Klik op het pictogram in de rechterbovenhoek.
    
3. Klik op **Site-instellingen**. Onder **Site-informatie**, klikt u op **deze site verwijderen**en klik vervolgens op **verwijderen**.
