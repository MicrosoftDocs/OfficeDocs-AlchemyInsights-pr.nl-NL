# <a name="project-for-the-web-common-issues-and-resolutions"></a>Project voor het web veelvoorkomende problemen en oplossingen

Hier volgen enkele populaire onderwerpen voor Project voor het web:

- Lees meer over [Project voor het web](https://support.microsoft.com/nl-NL/office/what-is-project-for-the-web-c19b2421-3c9d-4037-97c6-f66b6e1d2eb5), beschikbaar in alle [drie Project-abonnementen](https://products.office.com/project/compare-microsoft-project-management-software).
- Schakel Project voor het web of Roadmap [in of uit voor gebruikers in uw organisatie](https://docs.microsoft.com/project-for-the-web/turn-project-for-the-web-off). Beheerders kunnen de toegang tot Project voor het web beheren via de pagina **Projectinstellingen** in het Microsoft-beheercentrum.
- Meer informatie over [wat u kunt doen](https://support.office.com/article/what-can-you-do-with-project-for-the-web-b30f5442-be5f-43d2-9072-c95bff778ea1) met Project voor het web.
- Help gebruikers bij het maken en bouwen van hun projecten met [training voor Project voor het web](https://support.office.com/article/get-started-with-project-for-the-web-50bf3e29-0f0d-4b7a-9d2c-7c78389b67ad).
- Raak vertrouwd met de [limieten en grenzen](https://docs.microsoft.com/project-for-the-web/project-for-the-web-limits-and-boundaries) van Project voor het web.

Zie [Hulp voor beheerders van Project voor het web](https://docs.microsoft.com/project-for-the-web/projectforweb-admin-home) voor meer informatie over Project voor het web.
