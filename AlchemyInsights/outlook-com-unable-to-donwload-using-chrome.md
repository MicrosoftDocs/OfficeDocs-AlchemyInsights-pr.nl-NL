# <a name="cant-download-files-in-chrome"></a>Kan geen bestanden downloaden in Chrome

**Probleem** We zijn ons bewust van een probleem waardoor sommige klanten geen bestanden kunnen downloaden wanneer ze de Google Chrome-browser gebruiken. 

**Tijdelijke oplossing** Gebruik [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) of een andere webbrowser.
Meer informatie over het [downloaden van bestanden en Outlook.com bijlagen.](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy)
