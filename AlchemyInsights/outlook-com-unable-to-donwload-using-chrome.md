# <a name="cant-download-files-in-chrome"></a>Kan de bestanden in Chrome downloaden

**Probleem** We zijn op de hoogte van een probleem dat dat sommige klanten kunnen bestanden downloaden verhindert wanneer u Google Chrome browser. 

**Tijdelijke oplossing** [Microsoft Edge](https://www.microsoft.com/en-gb/windows/microsoft-edge) of een andere webbrowser gebruiken.
