# <a name="webresource-visibility-property"></a>Webresource Zichtbaarheid, eigenschap

U moet de **standaardoptie Zichtbaar selecteren** voor de webbron die op het formulier wordt weergegeven en beschikbaar is voor gebruikers. Zie [Webbron-eigenschappen voor modelgestuurde app-formulieren voor](https://docs.microsoft.com/powerapps/maker/model-driven-apps/web-resource-properties-legacy)meer informatie.
