# <a name="use-microsoft-edge-extensions"></a>Microsoft Edge-extensies gebruiken

[Extensies](https://go.microsoft.com/fwlink/?linkid=2135619) zijn invoegtoepassingen die u in Microsoft Edge kunt installeren om advertenties te blokkeren, wachtwoorden te beheren, tekst te vertalen en uw browse-ervaring op veel andere manieren aan te passen.
