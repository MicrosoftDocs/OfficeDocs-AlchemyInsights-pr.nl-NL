# <a name="microsoft-365-urls-and-ip-address-ranges"></a>URL's en IP-adresbereiken van Microsoft 365

Microsoft 365 vereist verbinding met internet via specifieke IP-adressen met *de naam eindpunten.*
Zie Url's en IP-adresbereiken van [Microsoft 365](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) voor een lijst met eindpunten die bereikbaar moeten zijn 

Met de microsoft 365 IP-adres- en URL-webservice kunt u microsoft 365-netwerkverkeer beter identificeren en onderscheiden, zodat u eenvoudiger wijzigingen kunt evalueren, configureren en op de hoogte kunt blijven. Zie [Microsoft 365 IP Address and URL web service](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
