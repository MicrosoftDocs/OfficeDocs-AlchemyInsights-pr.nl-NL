# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Url's en IP-adresbereiken voor Microsoft 365

Voor Microsoft 365 is connectiviteit met Internet vereist via specifieke IP-adressen met de naam *eindpunten*.
Zie [url's en IP-adresbereiken voor Microsoft 365](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) voor een lijst met eindpunten die bereikbaar moeten zijn. 

Met de Microsoft 365 IP Address and URL web service kunt u beter Microsoft 365-netwerkverkeer identificeren en onderscheiden, zodat u eenvoudiger de gewenste wijzigingen kunt evalueren, configureren en up-to-date blijven. Zie [Microsoft 365 IP Address and URL web service](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
