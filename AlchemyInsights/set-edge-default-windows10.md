# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>Microsoft Edge instellen als de standaardbrowser op een Windows 10-apparaat

Hier ziet u hoe dat gaat:

1. Selecteer de **knop Start** en voer **standaard-apps in.**

1. Selecteer standaard-apps in de **zoekresultaten.**

1. Selecteer **onder Webbrowser** de optie Microsoft **Edge.**
