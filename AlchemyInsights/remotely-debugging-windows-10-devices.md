# <a name="get-started-with-remotely-debugging-windows-10-devices"></a>Aan de slag met Windows 10-apparaten op afstand opsporen

U kunt als volgt de live-inhoud op een Windows 10-apparaat op afstand opsporen vanaf een Windows-of macOS-computer:

1. Stel uw Windows 10-apparaat in voor Remote Debugging en maak vervolgens verbinding met het apparaat vanaf de ontwikkelings computer.
2. U controleert en ontvangt live-inhoud op uw Windows 10-apparaat in de ontwikkelings computer.
3. Screencast inhoud van uw Windows 10-apparaat naar een DevTools-exemplaar op uw ontwikkelcomputer.

Voor meer informatie raadpleegt u [aan de slag met Remote Debugging Windows 10-apparaten met Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2142172).
