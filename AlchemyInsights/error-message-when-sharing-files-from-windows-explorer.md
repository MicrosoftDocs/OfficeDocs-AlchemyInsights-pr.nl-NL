# <a name="error-message-when-sharing-files-from-windows-explorer"></a>Foutbericht bij het delen van bestanden vanuit Windows Verkenner

Wanneer u probeert te delen vanuit Windows Verkenner, ontvangt u mogelijk het volgende foutbericht:
  
"Sorry, we kunnen de server nu niet bereiken. Probeer het later nog eens"
  
Als u een foutbericht wilt oplossen, gaat **Advanced** u vanuit Internet Explorer naar **Internetopties voor** \> **extra's** \> op het tabblad Geavanceerd en schakelt u het selectievakje **'Versleutelde pagina's niet opslaan op schijf' uit.**
  
 **Opmerking:** Mogelijk moet u uw browser opnieuw starten om de instelling van kracht te laten worden. Voor meer informatie zie [het foutbericht 'Kan de server nu niet bereiken' bij het delen van bestanden.](https://go.microsoft.com/fwlink/?linkid=2022914)
