# <a name="how-to-delete-a-folder-in-outlook-on-the-web"></a>Een map verwijderen in Outlook op het web

1. Klik in het mappenvenster met de rechtermuisknop op de map of submap die u wilt verwijderen.
2. Selecteer **map verwijderen**.
3. Selecteer **OK** om te bevestigen.

Zie [werken met berichtmappen in Outlook op het web](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f)voor meer informatie.
