# <a name="how-to-delete-a-folder-in-outlook-on-the-web"></a>Het verwijderen van een map in Outlook op het web

1. Klik in het mappenvenster met de rechtermuisknop op de map of submap die u wilt verwijderen.
2. Selecteer de **map verwijderen**.
3. Klik op **OK** om te bevestigen.

Voor meer informatie, Zie [werken met mappen in Outlook op het web weergegeven](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
