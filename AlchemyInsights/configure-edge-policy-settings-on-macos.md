# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Jamf gebruiken om Microsoft Edge-beleidsinstellingen te configureren voor macOS

Als u Microsoft Edge (stabiel kanaal 81 of hoger) wilt configureren voor macOS, gebruikt u een manifestbestand voor Microsoft Edge-beleid op Jamf Pro 10,19.

Zie [Microsoft Edge-beleidsinstellingen configureren op macOS met Jamf](https://go.microsoft.com/fwlink/?linkid=2134761)voor meer informatie.
