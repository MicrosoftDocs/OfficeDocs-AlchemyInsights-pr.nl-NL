# <a name="manage-office-installation-options"></a>Installatieopties voor Office beheren

Als Office 365-beheerder kunt u bepalen welke Office-software uw gebruikers van Office 365 kunnen downloaden en installeren. Zie [Manage Office installation options in the Microsoft 365 admin center](https://docs.microsoft.com/deployoffice/manage-software-download-settings-office-365) (Office-installatieopties beheren in het Microsoft 365-beheercentrum) voor meer informatie.
