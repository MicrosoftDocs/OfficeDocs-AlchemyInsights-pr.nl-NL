# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Een Windows 10-apparaat aan boord van Microsoft Defender Advanced Threat Protection (MDATP)

MDATP ondersteunt de volgende implementatiehulpmiddelen en -methoden voor Windows 10:

- Groepsbeleid
- Microsoft Endpoint Configuration Manager
- Mobile Device Management, inclusief Microsoft Intune
- Lokale scripts

Zie [Onboarding-hulpprogramma's en -methoden voor Windows 10-apparaten](https://go.microsoft.com/fwlink/?linkid=2143460)voor meer informatie.
