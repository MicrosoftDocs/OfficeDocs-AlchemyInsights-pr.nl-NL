# <a name="learn-how-microsoft-edge-supports-windows-information-protection-wip"></a>Ontdek hoe Microsoft EDGE ondersteuning biedt voor Windows Information Protection (OHW)

Wanneer het aantal externe medewerkers groter is, wordt het aantal persoonlijke en werkactiviteiten op bedrijfsapparaten gewijzigd. Hierdoor wordt het risico dat bedrijfsgegevens delen buiten de werkplek, groter weergeven. Microsoft Edge (versie 81 of hoger) ondersteunt Microsoft Edge (versie of hoger): een Windows-10-functie waarmee u ondernemingsgegevens kunt beschermen tegen niet-geautoriseerde en accidentele informatie.
