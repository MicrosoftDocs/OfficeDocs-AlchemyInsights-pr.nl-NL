# <a name="your-users-mailbox-is-being-created"></a>Het postvak van de gebruiker wordt gemaakt

Het kan een uur duren voordat een nieuw postvak kan worden gebruikt.
  
Als het postvak van de gebruiker gedurende een uur is voorbereid, kunt u proberen om de gebruiker te verwijderen en opnieuw te maken, of de optie ' We bellen u ' gebruiken om met een ondersteuningsmedewerker te praten.
