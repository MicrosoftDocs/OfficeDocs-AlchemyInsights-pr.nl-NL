# <a name="learn-about-the-security-and-privacy-features-of-microsoft-search-in-bing"></a>Meer informatie over de beveiligings- en privacyfuncties van Microsoft Search in Bing

De beveiligings- en privacyfuncties van Microsoft Search in Bing helpen een organisatie bij het beveiligen van de gebruikers- en werkplekgegevens. Aanvragen worden bijvoorbeeld via HTTPS gedaan en verbindingen worden van eind tot einde versleuteld.

Zie Beveiliging en privacy voor Microsoft Search in Bing voor [meer informatie.](https://docs.microsoft.com/microsoftsearch/security-for-search)
