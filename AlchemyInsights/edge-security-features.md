# <a name="learn-how-microsoft-edge-supports-identity-dependent-security-features"></a>Meer informatie Microsoft Edge identiteitsafhankelijke beveiligingsfuncties ondersteunt

Microsoft Edge (versie 77 of hoger) ondersteunt synchronisatie, eenmalige aanmelding (SSO) en andere functies die afhankelijk zijn van de identiteit van de gebruiker. Hierdoor kunnen gebruikers zich aanmelden via Active Directory Domain Services, Azure Active Directory en Microsoft-accounts.
