# <a name="use-microsoft-graph-api-to-manage-service-principal"></a>Microsoft Graph API gebruiken voor het beheren van de Service-Principal

Met de [Service-Principal-API](https://docs.microsoft.com/graph/api/resources/serviceprincipal)kunt u toepassingen beheren en bepalen wat een toepassing binnen uw Tenant kan doen.

Zie [toepassingen en Service-Principal-objecten in azure Active Directory](https://docs.microsoft.com/azure/active-directory/develop/app-objects-and-service-principals)voor meer informatie over de relatie tussen toepassingen en service-principals.
