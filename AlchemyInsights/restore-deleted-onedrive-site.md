# <a name="restore-a-deleted-onedrive-site"></a>Een verwijderde OneDrive-site terugzetten

Wanneer u een gebruiker verwijdert in het Microsoft 365-Beheercentrum (of wanneer een gebruiker wordt verwijderd via Active Directory-synchronisatie), wordt de OneDrive van de gebruiker bewaard voor het aantal dagen dat u opgeeft in het OneDrive-Beheercentrum. (Zie voor meer informatie [de standaard bewaaring voor bestanden van verwijderde OneDrive-gebruikers instellen](https://docs.microsoft.com/onedrive/set-retention).)

* [Een verwijderde OneDrive-site terugzetten](https://docs.microsoft.com/onedrive/restore-deleted-onedrive)

Als er veel van uw OneDrive-bestanden worden verwijderd, overschreven, beschadigd of besmet door malware, kunt u de volledige OneDrive herstellen voor een eerdere periode.

* [Uw OneDrive herstellen](https://support.office.com/article/Restore-your-OneDrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)
