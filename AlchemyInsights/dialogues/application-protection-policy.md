# <a name="application-protection-policy-requirements"></a>Vereisten voor toepassingsbeveiligingsbeleid

Vereisten voor toepassingsbeveiligingsbeleid:

- Gebruiker heeft een Intune- of EMS-licentie.
- Gebruiker behoort tot een groep die is gericht op toepassingsbeveiligingsbeleid.
- Slechts één zakelijke gebruiker is op een apparaat aangemeld bij beveiligde apps.
- [De toepassing heeft de Intune SDK geïmplementeerd. Zie beveiligde Microsoft Intune-apps voor een lijst met apps die de SDK ondersteunen.](https://docs.microsoft.com/mem/intune/apps/apps-supported-intune-apps)

[Meer informatie...](https://docs.microsoft.com/mem/intune/apps/app-protection-policy)
