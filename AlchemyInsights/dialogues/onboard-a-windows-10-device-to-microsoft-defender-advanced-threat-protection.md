# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Een Windows 10-apparaat in onboarden bij Microsoft Defender Advanced Threat Protection (MDATP)

MDATP ondersteunt de volgende implementatiehulpprogramma's en -methoden voor Windows 10:

- Groepsbeleid
- Microsoft Endpoint Configuration Manager
- Mobile Device Management, waaronder Microsoft Intune
- Lokale scripts

Zie [onboarding-hulpprogramma's en -methoden voor Windows 10-apparaten](https://go.microsoft.com/fwlink/?linkid=2143460)voor meer informatie.
