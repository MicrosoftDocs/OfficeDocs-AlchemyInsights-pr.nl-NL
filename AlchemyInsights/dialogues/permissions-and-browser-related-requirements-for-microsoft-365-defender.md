# <a name="permissions-and-browser-related-requirements-for-microsoft-365-defender"></a>Machtigingen en browservereisten voor Microsoft 365 Defender

- Machtigingen: Als u Microsoft 365 Defender wilt in dienst gaan, moet u een globale beheerder of een beveiligingsbeheerder zijn in Azure Active Directory. Zie Toegang tot [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2143626)beheren voor een lijst met rollen die Microsoft 365 Defender kunnen gebruiken en als u wilt weten hoe toegang tot gegevens is geregeld.

- Browservereisten: Voor toegang tot Microsoft 365 Defender in het Microsoft 365-beveiligingscentrum gebruikt u Microsoft Edge, Internet Explorer 11 of een webbrowser die compatibel is met HTML 5.
