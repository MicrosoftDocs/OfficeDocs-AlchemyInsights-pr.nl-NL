# <a name="increase-onedrive-storage"></a>OneDrive-opslag verhogen

Als globale beheerder of als SharePoint-beheerder in Microsoft 365, kunt u de OneDrive-opslagruimte voor een specifieke gebruiker rechtstreeks vanuit het Microsoft-Beheercentrum instellen.  

Zie [de OneDrive-opslagruimte van een bepaalde gebruiker wijzigen](https://docs.microsoft.com/onedrive/change-user-storage)voor meer informatie.
