# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Een uitbreiding aan Microsoft Edge toevoegen vanuit de Chrome Web Store

Voer de volgende stappen uit om een uitbreiding toe te voegen aan de Microsoft Edge in de Chrome Web Store:

1. Ga naar de Chrome Web Store.
2. Selecteer in de banner boven aan de pagina de optie **extensies uit andere winkels toestaan**.
3. Selecteer de extensie die u wilt toevoegen en klik op **toevoegen aan Chrome**.
4. Controleer in het bevestigingsvenster de aangebrachte wijzigingen aandachtig door en klik op **extensie toevoegen**.
In het laatste dialoogvenster wordt bevestigd dat de extensie is toegevoegd.
