# <a name="manage-groups-in-yammer"></a>Groepen beheren in Yammer

- **Machtigingen voor het maken van groepen.** Yammer beheert niet wie een groep kan maken. U kunt ook het maken van groepen voor verbonden Microsoft 365-groepen beheren. Zie [Manage wie Microsoft 365-groepen kunnen maken](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups) voor meer informatie.

- **Dynamisch groepslidmaatschap.** Raadpleeg [een dynamische groep in Yammer maken](https://docs.microsoft.com/yammer/manage-yammer-groups/create-a-dynamic-group) voor informatie over het maken van dynamische lidmaatschap voor Yammer-groepen met microsoft 365.

- **Verwijderde groepen herstellen.** Yammer-groepen herstellen is mogelijk via [het terugzetten van een verwijderde Microsoft 365-groep](https://docs.microsoft.com/microsoft-365/admin/create-groups/restore-deleted-group) niet-verbonden groepen kan na verwijdering niet worden hersteld.

Zie voor meer informatie over het beheren van een groep [een groep beheren in Yammer](https://support.office.com/article/Manage-a-group-in-Yammer-6e05c6d6-5548-4c88-89cd-e6757a514ef2). Zie [Yammer-en Microsoft 365-groepen](https://docs.microsoft.com/yammer/manage-yammer-groups/yammer-and-office-365-groups)voor meer informatie over verbonden microsoft 365-groepen.
