# <a name="using-the-microsoft-authenticator-app"></a>Met de Microsoft Authenticator-app

Meld u aan bij uw persoonlijke Microsoft-account met een vingerafdruk, gezichtsherkenning of een pincode.

- [De Microsoft Authenticator-app gebruiken](https://support.microsoft.com/help/4026727/microsoft-account-how-to-use-the-microsoft-authenticator-app). 

- U hoeft [de Authenticator-app](https://docs.microsoft.com/azure/active-directory/user-help/security-info-setup-auth-app) in te stellen door de QR-code te scannen of door [handmatig een account toe te voegen](https://docs.microsoft.com/azure/active-directory/user-help/user-help-auth-app-add-account-manual).  

Zie het overzicht van de [verificator-app](https://docs.microsoft.com/azure/active-directory/user-help/user-help-auth-app-overview)voor meer informatie.
