# <a name="restore-deleted-users-and-groups"></a>Verwijderde gebruikers en groepen terugzetten

Als u een gebruiker per ongeluk hebt verwijderd, raadpleegt u [een onlangs verwijderde gebruiker herstellen of verwijderen via Azure Active Directory](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-users-restore?context=azure/active-directory/users-groups-roles/context/ugr-context).

Zie [een verwijderde groep Microsoft 365 herstellen in azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/groups-restore-deleted)als u de verwijderde groep wilt herstellen.
