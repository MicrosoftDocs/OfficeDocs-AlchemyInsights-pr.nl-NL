# <a name="enable-or-disable-forms-pro"></a>Formulieren Pro in- of uitschakelen

Als u Microsoft Forms Pro probeert in te schakelen, wordt deze opgenomen als onderdeel van de [Dynamics 365](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant) enterprise-licentie. Als alternatief kan een beheerder een [Microsoft Forms Pro-licentie](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant)aanschaffen. U [ook een Microsoft Forms Pro-licentie van](https://docs.microsoft.com/forms-pro/purchase#disable-microsoft-forms-pro-for-a-user-1) een gebruiker verwijderen.
