# <a name="create-a-new-folder"></a>Maak een nieuwe map

1. Selecteer de **nieuwe map**onder **mappen** in het navigatiedeelvenster.

2. Typ een naam voor de map en druk op Enter.

Zie voor meer informatie, [een map in Outlook.com maken](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).
