# <a name="conditional-access"></a>Voorwaardelijke toegang

Voorwaardelijke toegang is een mogelijkheid van Azure Active Directory. Met voorwaardelijke toegang kunt u automatisch toegangsbeheer voor toegangsbeheer implementeren voor het openen van Cloud-apps die op voorwaarden zijn gebaseerd. Voor meer informatie gaat u naar [op locatie beleid configureren](https://docs.microsoft.com/azure/active-directory/conditional-access/overview).

Als IT-beheerder kunt u ook toegang tot SharePoint-en OneDrive-bronnen beheren op basis van gedefinieerde netwerklocaties of apparaten die u vertrouwt. Ga naar deze links voor meer informatie.

- [Toegang tot SharePoint Online en OneDrive-gegevens regelen op basis van netwerklocatie](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location)

- [Toegang beheren vanaf niet-beheerde apparaten](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)
