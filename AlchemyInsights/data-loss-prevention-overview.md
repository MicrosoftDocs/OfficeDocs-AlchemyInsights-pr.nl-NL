# <a name="data-loss-prevention-dlp-overview"></a>Overzicht van Data Loss Prevention (DLP)

Met een DLP-beleid u:

- Identificeer gevoelige informatie op veel locaties, zoals Exchange Online, SharePoint Online en OneDrive voor Bedrijven.


U bijvoorbeeld elk document identificeren dat een creditcardnummer bevat dat is opgeslagen op een OneDrive voor Bedrijven-site, of u alleen de OneDrive-sites van specifieke personen controleren.

- Voorkom het per ongeluk delen van gevoelige informatie.


Voor meer informatie u het volgende [artikel](https://docs.microsoft.com/microsoft-365/compliance/data-loss-prevention-policies)bekijken.
