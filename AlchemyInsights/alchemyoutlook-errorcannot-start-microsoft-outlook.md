# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Outlook-foutbericht: 'Kan Microsoft Outlook niet starten'

Zijn er problemen met het starten van Outlook 2016, Outlook 2013 of Outlook 2010 of ontvangst 'kan Microsoft Outlook niet starten. Kan het Outlook-venster niet openen' foutberichten?
  
Gebruik eerst de stappen voor probleemoplossing in [dit artikel](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) voor het oplossen van deze problemen is gestart. 
  
U kunt ook het [hulpprogramma geavanceerde diagnostische gegevens](https://aka.ms/SaRA-OutlookAdvDiagnostics) uitvoeren op de computer waarin dit probleem optreedt.
