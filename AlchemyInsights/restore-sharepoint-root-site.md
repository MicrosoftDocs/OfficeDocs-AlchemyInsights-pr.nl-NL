# <a name="restore-the-sharepoint-root-site"></a>De SharePoint-hoofdsite herstellen

Herstel de verwijderde hoofdsite vanuit het nieuwe SharePoint-beheercentrum. Zodra de site is hersteld, vervangt u deze in plaats daarvan.

1. Ga naar [de pagina Verwijderde sites.](https://admin.microsoft.com/sharepoint?page=recycleBin&modern=true) 
2. Selecteer de hoofdsite die u wilt herstellen.
3. Klik op **Terugzetten**.
4. [Vervang uw hoofdsite.](https://docs.microsoft.com/sharepoint/troubleshoot/sites/url-that-resides-under-root-site-collection-is-broken)

Zie De hoofdsite moderniseren [voor meer informatie.](https://docs.microsoft.com/sharepoint/modern-root-site)
