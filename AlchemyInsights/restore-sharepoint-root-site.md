# <a name="restore-the-sharepoint-root-site"></a>De Hoofdsite van SharePoint herstellen

Herstel de verwijderde hoofdsite uit het nieuwe SharePoint-beheercentrum. Wanneer deze is hersteld, vervangt u de site in plaats daarvan.

1. Ga naar [de pagina Verwijderde sites.](https://admin.microsoft.com/sharepoint?page=recycleBin&modern=true) 
2. Selecteer de hoofdsite die u wilt herstellen.
3. Klik op **Terugzetten**.
4. [Vervang uw hoofdsite.](https://docs.microsoft.com/sharepoint/troubleshoot/sites/url-that-resides-under-root-site-collection-is-broken)

Zie Uw hoofdsite moderniseren voor [meer informatie.](https://docs.microsoft.com/sharepoint/modern-root-site)
