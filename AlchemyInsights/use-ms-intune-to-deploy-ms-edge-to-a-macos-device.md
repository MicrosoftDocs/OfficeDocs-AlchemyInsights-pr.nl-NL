# <a name="use-microsoft-intune-to-deploy-microsoft-edge-to-a-macos-device"></a>Microsoft intune gebruiken om Microsoft Edge te implementeren op een macOS-apparaat

Als u een app wilt implementeren, configureren, beheren en beveiligen op een macOS-apparaat, moet u de app toevoegen aan Microsoft intune. De app moet van een type worden ondersteund door intune en de beschikbare beheeropties voor intune zijn afhankelijk van het type van de app. Met intune hoeft u ook niet het macOS-hulpprogramma voor de app-terugloop te gebruiken.

InTune ondersteunt Microsoft Edge. Zie [Microsoft Edge toevoegen aan macOS-apparaten met behulp van Microsoft intune](https://go.microsoft.com/fwlink/?linkid=2134949)voor meer informatie.
