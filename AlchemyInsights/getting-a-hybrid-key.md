# <a name="getting-an-exchange-hybrid-key"></a>Een hybride Exchange-sleutel opdoen

Als u uw on-premises organisatie wilt verbinden met Exchange Online in Microsoft 365, moet u ten minste één Exchange-server in uw on-premises organisatie instellen. Deze server wordt gebruikt om een hybride implementatie te maken die uw on-premises Exchange-organisatie verbindt met Exchange Online. Als u een hybride sleutel voor deze server wilt krijgen, voert u de stappen uit bij de [wizard Hybride productdistributie van Exchange.](https://aka.ms/hybridkey)
