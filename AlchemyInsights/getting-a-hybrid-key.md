# <a name="getting-an-exchange-hybrid-key"></a>Een hybride Exchange-sleutel verkrijgen

Als u uw on-premises organisatie wilt verbinden met Exchange Online in Microsoft 365, moet u ten minste één Exchange-Server instellen in uw on-premises organisatie. Deze server wordt gebruikt voor het maken van een hybride implementatie waarmee uw on-premises Exchange-organisatie wordt verbonden met Exchange Online. Als u een hybride sleutel voor deze server wilt gebruiken, volgt u de stappen in de [wizard hybride versie van productcode voor Exchange](https://aka.ms/hybridkey).
