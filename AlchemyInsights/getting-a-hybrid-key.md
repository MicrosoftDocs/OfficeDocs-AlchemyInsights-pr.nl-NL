# <a name="getting-an-exchange-hybrid-key"></a>Een hybride Exchange sleutel ophalen

Voor uw organisatie op locatie verbinding met Exchange Online in Office 365, moet u ten minste één Exchange server in uw organisatie op locatie instellen. Deze server wordt gebruikt voor het maken van een hybride distributie die uw Exchange-organisatie op ruimten met Exchange Online verbonden. Als u een hybride-sleutel voor deze server, volg de stappen op [Exchange hybride product key distribution wizard](https://aka.ms/hybridkey).
