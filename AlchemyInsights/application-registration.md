# <a name="application-registration"></a>Sollicitatie registratie

1. [Een app registreren met Azure Active Directory](https://docs.microsoft.com/powerapps/developer/data-platform/walkthrough-register-app-azure-active-directory): in deze zelfstudie wordt beschreven hoe u een toepassing registreert met Azure Active Directory, waarmee een gebruiker met een Power Point-gebruikersaccount verbinding kan maken met hun Microsoft Dataverse-omgeving via externe clienttoepassingen via OAuth-verificatie.
1. [Een toepassing registreren met Microsoft Identity platform](https://docs.microsoft.com/azure/active-directory/develop/quickstart-register-app): in deze handleiding registreert u een app in de Azure-Portal, zodat het Microsoft Identity platform verificatie-en autorisatie Services kan verlenen voor uw toepassing en de gebruikers.
