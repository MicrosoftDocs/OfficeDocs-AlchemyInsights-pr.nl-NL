# <a name="print-results-in-a-summary-of-responses"></a>Resultaten in een samenvatting van de antwoorden afdrukken

U kunt [Microsoft Forms resultaten afdrukken](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc) als een samenvatting van de antwoorden of voor een afzonderlijke. 

Formulieren zonder antwoorden kan niet worden afgedrukt, maar u kunt aanvragen of stemmen over deze functie via de [site van Microsoft Forms UserVoice](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box).
