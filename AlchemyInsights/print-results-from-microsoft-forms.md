# <a name="print-results-in-a-summary-of-responses"></a>Resultaten in een overzicht van de antwoorden afdrukken

U kunt [resultaten van Microsoft Forms afdrukken](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc) als een overzicht van de antwoorden of voor een individu. 

U kunt geen formulieren afdrukken zonder antwoorden, maar u kunt deze functie wel aanvragen of stemmen op de [Bwebsite Microsoft Forms UserVoice](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box).
