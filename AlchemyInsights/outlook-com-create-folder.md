# <a name="create-a-folder-in-outlookcom"></a>Maak een map in Outlook.com

1. Selecteer de **nieuwe map**onderaan in het deelvenster met mappen. U moet mogelijk naar beneden schuiven in het mappenvenster op de knop **nieuwe map** wordt weergegeven.
2. Voer een naam voor de map in het tekstvak en druk op Enter.

Zie voor meer informatie, [een map in Outlook.com maken](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
