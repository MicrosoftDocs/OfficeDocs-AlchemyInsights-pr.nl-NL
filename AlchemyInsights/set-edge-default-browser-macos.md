# <a name="set-microsoft-edge-as-the-default-browser-on-a-macos-device"></a>Microsoft Edge instellen als de standaardbrowser op een macOS-apparaat

Gebruik een van deze twee methoden om Microsoft Edge in te stellen als de standaardbrowser:

Methode 1: Knipper het apparaat met een afbeelding van macOS waarin Microsoft Edge al is ingesteld als de standaardbrowser.

Methode 2: Stel het beleid DefaultBrowserSettingEnabled in om de gebruiker te vragen Microsoft Edge in te stellen als de standaardbrowser.

Met beide methoden kan een gebruiker de standaardbrowser wijzigen. Daarom wordt u aangeraden het beleid DefaultBrowserSettingEnabled te implementeren, zelfs als u methode 1 hebt gebruikt.
