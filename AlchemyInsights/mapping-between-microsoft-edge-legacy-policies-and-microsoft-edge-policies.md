# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Meer informatie over het toewijzen van Microsoft Edge Legacy-beleid en Microsoft Edge-beleid

De groepsbeleidsregels van Microsoft Edge Legacy (versies 45 en eerder) en de MDM-instellingen (Mobile Device Management) worden in kaart gebracht aan het relevante beleid van Microsoft Edge, versie 80. Zie Microsoft [Edge Legacy to Microsoft Edge policy mapping voor meer informatie.](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge)
