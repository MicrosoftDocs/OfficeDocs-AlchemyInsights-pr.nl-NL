# <a name="create-users-in-hybrid-deployments"></a>Gebruikers maken in hybride implementaties

Als uw organisatie een hybride implementatie heeft tussen een on-premises omgeving en Microsoft 365, moet u alle gebruikersobjecten maken in uw on-premises Active Directory. U maakt de gebruikersobjecten in de juiste organisatie-eenheid (OU) zodat ze worden gesynchroniseerd met uw organisatie.
