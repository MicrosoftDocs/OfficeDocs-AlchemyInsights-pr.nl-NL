# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Een lijst met toegestane eindpunten voor Microsoft Edge maken

Als u de volledige array van de functies van Microsoft Edge wilt gebruiken voor de verwijdering, moet u deze verbinding hebben met internet. Als u wilt dat de browser communiceert via firewalls en andere beveiligingsmechanismen op internet, moet u een reeks domein-Url's toevoegen aan de [lijst toestaan van eindpunten](https://go.microsoft.com/fwlink/?linkid=2135054).
