# <a name="manage-who-can-create-microsoft-365-groups"></a>Beheren wie Microsoft 365-groepen kunnen maken

Standaard kan iedereen in uw organisatie O365-groepen maken.
  
Als u dit wilt wijzigen, kunt u een beveiligingsgroep met PowerShell maken die het maken van groepen beperkt tot een subset van gebruikers.
  
Volg de stapsgewijze instructies hier: [beheren wie Microsoft 365-groepen kan maken](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
