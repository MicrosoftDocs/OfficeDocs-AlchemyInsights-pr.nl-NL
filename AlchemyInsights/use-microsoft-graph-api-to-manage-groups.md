# <a name="use-microsoft-graph-api-to-manage-groups"></a>Microsoft Graph API gebruiken om groepen te beheren

Groepen zijn verzamelingen gebruikers en andere principals die toegang hebben tot bronnen in Microsoft-services of in de app. Microsoft Graph biedt Api's die u kunt gebruiken voor het maken en beheren van verschillende typen groepen en groeps functionaliteit overeenkomstig uw scenario. Voor alle bewerkingen die aan de groep zijn gekoppeld in Microsoft Graph, moet u een beheerder toestemming verlenen. Zie [werken met groepen in Microsoft Graph](https://docs.microsoft.com/graph/api/resources/groups-overview)voor meer informatie.
