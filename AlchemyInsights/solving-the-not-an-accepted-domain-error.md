# <a name="got-a-parameter-cannot-be-found-error"></a>Heb je een fout 'Parameter kan niet gevonden worden'?

Als u het foutbericht **'Een parameter kan niet vinden die overeenkomt met de parameternaam 'E-mailadressen',** betekent dit dat het iets langer duurt voordat Microsoft 365 klaar is met het instellen van uw tenant of uw aangepaste domein als u er onlangs een hebt toegevoegd.
  
Het instellen kan 4 uur duren. Wacht even, zodat de instellingsprocedure kan worden voltooid en probeer het vervolgens opnieuw.
