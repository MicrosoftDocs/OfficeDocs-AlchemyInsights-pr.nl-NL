# <a name="got-a-parameter-cannot-be-found-error"></a>De fout ' parameter kan niet worden gevonden ' wordt weergegeven?

Als het foutbericht **' er kan geen parameter worden gevonden die overeenkomt met de naam van de parameter '** wordt weergegeven, betekent dit dat het langer duurt voor microsoft 365 om het instellen van uw Tenant of uw aangepaste domein te voltooien als u onlangs een naam hebt toegevoegd.
  
Het instellen kan 4 uur duren. Wacht even, zodat de instellingsprocedure kan worden voltooid en probeer het vervolgens opnieuw.
