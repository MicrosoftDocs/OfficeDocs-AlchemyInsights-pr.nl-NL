# <a name="limit-sharing-in-microsoft-365"></a>Delen beperken in Microsoft 365

Hoewel u intern delen helemaal niet kunt uitschakelen of de knop delen van sites kunt verwijderen, kunt u op verschillende manieren beperkingen instellen voor het delen van bestanden in Microsoft 365 met de behoeften van uw organisatie. Zie [beperkingen voordelen in Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)voor meer informatie.
