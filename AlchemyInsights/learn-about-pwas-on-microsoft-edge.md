# <a name="learn-about-the-progressive-web-apps-pwas-on-microsoft-edge-chromium"></a>Meer informatie over de Progressive Web Apps (PWAs) op Microsoft Edge (Chromium)

Progressive Web Apps (PWAs) zijn websites die geleidelijk worden verbeterd om te functioneren als native apps op platforms die ze ondersteunen. PBO's combineren de beste eigenschappen van web- en native apps.

Vanuit het perspectief van een webplatform zijn [P PWA's op Microsoft Edge (Chromium)](https://docs.microsoft.com/microsoft-edge/progressive-web-apps-chromium/#pwas-on-microsoft-edge-chromium) gebaseerd op standaarden. Gebruikers kunnen apps rechtstreeks vanuit de browser installeren en ze hebben geen implementatie of registratie op basis van de Microsoft Store nodig. Desktop PWAs worden ondersteund op elk platform waar Microsoft Edge (Chromium) beschikbaar is.
