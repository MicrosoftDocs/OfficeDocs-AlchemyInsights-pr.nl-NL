# <a name="restrict-commenting-in-microsoft-stream"></a>Opmerkingen beperken in Microsoft Stream

Als organisatie wilt u mogelijk opmerkingen uitschakelen voor alle video's die in Stream worden geplaatst. Standaard kunnen opmerkingen op de video's worden in- of uitgeschakeld.

- [Opmerkingen beperken in Microsoft Stream](https://docs.microsoft.com/stream/portal-disable-comments)
