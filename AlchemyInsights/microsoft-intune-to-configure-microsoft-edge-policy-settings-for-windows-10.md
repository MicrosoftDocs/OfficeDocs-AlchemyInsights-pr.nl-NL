# <a name="use-microsoft-intune-to-configure-microsoft-edge-policy-settings-for-windows-10"></a>Microsoft Intune gebruiken om beleidsinstellingen voor Microsoft Edge voor Windows 10 te configureren

U kunt Microsoft Edge-beleid en -instellingen configureren door een apparaatconfiguratieprofiel toe te voegen aan Microsoft Intune. Dit komt overeen met het configureren van Groepsbeleidsregels voor Microsoft Azure Active Directory of lokale groepsbeleidsobjectinstellingen (GPO) op gebruikersapparaten.

Zie Een profiel maken voor het beheren [van instellingen in Microsoft Edge voor Windows 10](https://go.microsoft.com/fwlink/?linkid=2133700)voor meer informatie.

Dit artikel is van toepassing op Microsoft Edge-versies 77 en hoger.
