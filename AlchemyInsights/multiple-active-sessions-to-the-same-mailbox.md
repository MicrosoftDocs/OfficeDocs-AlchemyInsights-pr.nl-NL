# <a name="multiple-active-sessions-to-the-same-mailbox"></a>Meerdere actieve sessies naar hetzelfde postvak

Als u het gebruik van Exchange-resources wilt beheren, heeft een postvak een 'budget'.

De uitzondering over het budget kan worden geactiveerd door, maar is niet beperkt tot, de volgende omstandigheden:

- In dezelfde Outlook Web App-sessie worden enkele browsertabbladen geopend.

- Een paar actieve Outlook Web App-sessies naar hetzelfde postvak.

- Een paar andere clienttoepassingen (Outlook, Outlook Mobile, een client-app van derden) hebben tegelijkertijd toegang tot het postvak.

- Langlopende bewerkingen, zoals het uitvoeren van zoekverzoeken, worden uitgevoerd vanuit een andere actieve postvaksessie.
