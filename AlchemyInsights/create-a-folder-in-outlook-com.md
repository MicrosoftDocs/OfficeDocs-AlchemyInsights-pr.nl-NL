# <a name="create-a-folder"></a>Een map maken

1. Selecteer de **Nieuwe map maken** bij **mappen**in het linkerdeelvenster. 
    
2. In de nieuwe map, typ een naam voor de map en druk op Enter.
    
Zie voor meer informatie, [een map in Outlook.com maken](https://go.microsoft.com/fwlink/p/?linkid=873114).
