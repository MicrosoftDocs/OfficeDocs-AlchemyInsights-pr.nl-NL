# <a name="microsoft-stream"></a>Microsoft Stream

Microsoft Stream is een zakelijke videoservice waarmee personen in een organisatie veilig video's kunnen uploaden, weergeven en delen. U kunt opnamen delen van lessen, vergaderingen, presentaties trainingssessies of andere video's die de samenwerking in uw team helpen.  

- [De nieuwe Microsoft stream](https://docs.microsoft.com/stream/new-stream)
- [Microsoft stream (klassiek)](https://docs.microsoft.com/stream/overview)
- [Migreren van Microsoft stream (klassiek) naar de nieuwe Microsoft stream](https://docs.microsoft.com/stream/classic-migration)
