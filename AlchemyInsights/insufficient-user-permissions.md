# <a name="insufficient-user-permissions"></a>Onvoldoende gebruikersmachtigingen

Dit probleem doet zich voor als er niet aan de benodigde [vereisten](https://docs.microsoft.com/configmgr/tenant-attach/device-sync-actions#prerequisites) wordt voldaan. Zorg ervoor dat de gebruiker toegang heeft tot **Bron waarschuwen** onder objectklasse **Collecties** in Configuration Manager.
