# <a name="microsoft-stream-storage"></a>Microsoft Stream-opslag

Microsoft Stream-tenants ontvangen een vaste toewijzing van 500 GB opslagruimte en een extra 0,5 GB opslagruimte per gebruiker met een licentie.
Zie voor meer informatie over opslag:

- [Overzicht van Microsoft Stream-opslag](https://docs.microsoft.com/stream/license-overview#storage)
- [Opslag toevoegen aan Microsoft Stream](https://docs.microsoft.com/stream/storage-add-on)
- [Quota en beperkingen in Microsoft Stream](https://docs.microsoft.com/stream/quotas-and-limitations)
