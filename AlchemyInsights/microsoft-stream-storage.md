# <a name="microsoft-stream-storage"></a>Opslag voor Microsoft stream

Microsoft stream-tenants ontvangen een vaste toewijzing van 500 GB opslagruimte en een extra 0,5 GB opslagruimte per gelicentieerde gebruiker.
Zie voor meer informatie over opslag:

- [Overzicht van Microsoft stream-opslag](https://docs.microsoft.com/stream/license-overview#storage)
- [Opslag toevoegen aan Microsoft stream](https://docs.microsoft.com/stream/storage-add-on)
- [Quota's en beperkingen in Microsoft stream](https://docs.microsoft.com/stream/quotas-and-limitations)
