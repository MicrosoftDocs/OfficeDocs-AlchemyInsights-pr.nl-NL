# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Voorwaarden of geavanceerde formules gebruiken in een stroom

U kunt in Microsoft flow een stroom aanpassen, zodat de taken alleen worden uitgevoerd als een voorwaarde waar is. Zie [een voorwaarde toevoegen aan een flow](https://go.microsoft.com/fwlink/?linkid=872112)voor informatie.
  
Als u een geavanceerde formule wilt maken, raadpleegt u [taal schema voor werkstroomdefinities voor Azure Logic-apps](https://aka.ms/logicexpressions).
