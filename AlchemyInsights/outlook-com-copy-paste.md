# <a name="copy-and-paste-in-outlookcom"></a>Kopiëren en plakken in Outlook.com

De knoppen kopiëren en plakken worden niet in Outlook.com gebruikt, maar u kunt wel toetsenbordsneltoetsen gebruiken:

**Op een PC:** Druk op CTRL + C om te kopiëren, CTRL + X om te knippen en CTRL + V om te plakken.

**Op een Mac:** CMD + C om te kopiëren, CMD + X om te knippen en CMD + V om te plakken.
